# Pomodoro Study Timer - Quick Start Guide

## 🚀 Getting Started in 3 Steps

### 1. Navigate to Timer
- Log in as a **student**
- Click **"Pomodoro Timer"** in the sidebar (marked with "NEW" badge)
- Or go to: `http://localhost:5173/student/pomodoro`

### 2. Start Your First Session
```
1. (Optional) Select a subject from the dropdown
2. Click the big "Start" button
3. Focus on studying for 25 minutes
4. Take a break when the timer completes
```

### 3. View Your Progress
```
1. Click the "Analytics" tab
2. See your study statistics and charts
3. Track your consistency with the streak calendar
```

## ⚙️ Default Settings

```yaml
Work Duration: 25 minutes
Short Break: 5 minutes
Long Break: 15 minutes
Long Break After: 4 sessions
Auto-start: Off
Notifications: On (if permitted)
```

## 🎨 Color Guide

- 🔵 **Blue** = Work session (focus time)
- 🟢 **Green** = Short break (5 min)
- 🔵 **Light Blue** = Long break (15 min)

## 🔧 Common Issues

### Timer won't start?
- Check if you're on the Timer tab
- Ensure timer shows "Idle" state
- Refresh page if needed

### No notifications?
- Check browser permissions
- Enable in Settings dialog
- Ensure sound is on

### Analytics empty?
- Complete at least one session first
- Check date range
- Try refresh button

## 🎓 Study Technique

### The Pomodoro Method
```
1. Choose task to focus on
2. Set timer for 25 minutes
3. Work without interruption
4. Take 5-minute break
5. Repeat 4 times
6. Take 15-minute long break
```

### Why It Works
- ✓ Prevents burnout
- ✓ Maintains focus
- ✓ Regular rest periods
- ✓ Builds discipline
- ✓ Tracks progress
