# Quick Start Guide

## Generate Presentation in 3 Steps

### Windows Users

1. **Navigate to presentation folder:**
   ```powershell
   cd presentation
   ```

2. **Run the generator:**
   ```powershell
   .\generate.ps1
   ```

3. **Open the file:**
   ```
   Educational_SaaS_Platform_Presentation.pptx
   ```

### Linux/Mac Users

1. **Navigate to presentation folder:**
   ```bash
   cd presentation
   ```

2. **Run the generator:**
   ```bash
   chmod +x generate.sh
   ./generate.sh
   ```

3. **Open the file:**
   ```
   Educational_SaaS_Platform_Presentation.pptx
   ```
