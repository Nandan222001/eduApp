================================================================================
WHAT THE SCRIPTS DO
================================================================================

1. Remove node_modules/ directory (handles Windows long paths)
2. Remove .expo/ directory (Expo build cache)
3. Delete package-lock.json
4. Clear npm cache (npm cache clean --force)
5. Reinstall dependencies (npm install)
6. Start Expo with clear cache (npx expo start --clear)

Time Required: 5-10 minutes (depending on internet speed)
