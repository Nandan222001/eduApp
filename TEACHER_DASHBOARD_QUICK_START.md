# Teacher Dashboard - Quick Start Guide

## Overview
The Teacher Dashboard is a comprehensive interface designed for teachers to manage their classes, assignments, and student interactions efficiently.

## Access the Dashboard

1. **Login as a Teacher**
   - Navigate to `/login`
   - Enter teacher credentials
   - Verify email if required

2. **Dashboard Access**
   - Automatic redirect to `/teacher` or `/teacher/dashboard`
   - Available in the main navigation menu

## Dashboard Sections

### 1. Statistics Cards (Top Row)
- **Total Students**: Shows count of all students across your classes
- **Pending Grading**: Number of assignments waiting to be graded
- **Today's Classes**: Number of scheduled periods today
- **This Week's Attendance**: Average attendance percentage

### 2. Quick Actions (Header)
Three primary buttons for common tasks:
- **Mark Attendance**: Click to mark student attendance
- **Create Assignment**: Click to create a new assignment
- **Post Announcement**: Click to post announcements to classes

### 3. My Classes Overview
- Grid of all your assigned classes
- Click any class card to view details
- Shows: Class name, section, subject, student count, average score, room number

### 4. Today's Schedule
- Displays your schedule for today
- Shows time slots, class names, subjects, and room numbers
- Color-coded status badges:
  - **Green**: Completed
  - **Orange**: Ongoing
  - **Blue**: Upcoming

### 5. Pending Grading
- Table of assignments awaiting your review
- Shows submission count and due dates
- Priority indicator on the left:
  - **Red**: High priority (overdue)
  - **Orange**: Medium priority
  - **Blue**: Low priority
- Click any row to start grading

### 6. Recent Student Submissions
- Feed of latest student submissions
- Click any submission to review details
- Shows grading status and scores

### 7. Class Performance Charts
- **Bar Chart**: Compare scores and attendance across classes
- **Doughnut Chart**: Weekly attendance percentage visualization

### 8. Upcoming Exams
- Grid of scheduled examinations
- Shows exam details: name, type, date, duration, marks
- Click any exam card for more details

## Common Tasks

### Mark Attendance
1. Click "Mark Attendance" button in header
2. Select class and date
3. Mark students as present/absent/late
4. Save attendance record

### Create Assignment
1. Click "Create Assignment" button
2. Fill in assignment details
3. Set due date and total marks
4. Assign to classes
5. Submit

### Grade Assignments
1. Navigate to "Pending Grading" section
2. Click on assignment to grade
3. Review student submissions
4. Enter scores and feedback
5. Save grades

### Post Announcement
1. Click "Post Announcement" button
2. Write announcement message
3. Select target classes
4. Set priority (optional)
5. Post to students

## Tips & Tricks

### Navigation
- Use breadcrumbs for easy navigation
- Click on cards and rows for quick access
- Use browser back button to return

### Filtering
- Use search bars to find specific items
- Filter by date ranges (where available)
- Sort tables by clicking column headers

### Performance
- Dashboard refreshes automatically
- Pull down to refresh on mobile
- Charts update in real-time

### Keyboard Shortcuts
- `Ctrl/Cmd + K`: Quick search
- `Esc`: Close dialogs/modals
- `Tab`: Navigate between elements

## Mobile Access

The dashboard is fully responsive:
- Cards stack vertically on mobile
- Tables scroll horizontally
- Touch-friendly buttons and controls
- Optimized for smaller screens

## Troubleshooting

### Dashboard Not Loading
- Check internet connection
- Refresh the page
- Clear browser cache
- Verify login status

### Missing Data
- Ensure you have classes assigned
- Check date filters
- Contact admin if classes not showing

### Charts Not Displaying
- Update browser to latest version
- Enable JavaScript
- Disable ad blockers temporarily

### Performance Issues
- Close unused browser tabs
- Clear browser cache
- Use modern browser (Chrome, Firefox, Edge)
