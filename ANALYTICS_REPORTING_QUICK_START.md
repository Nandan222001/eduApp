# Analytics and Reporting UI - Quick Start Guide

This guide helps you quickly get started with the analytics and reporting features.

## For Students

### Accessing Student Analytics

1. Log in as a student
2. Navigate to **Dashboard** → **Analytics** or visit `/student/analytics`
3. View your comprehensive performance dashboard

### Available Charts

**Subject Performance Trends**
- Multi-line chart showing your scores across all subjects over time
- Compare your performance to identify strengths and weaknesses

**Attendance Heatmap**
- Calendar view of your attendance patterns
- Color-coded: Green (Present), Red (Absent), Yellow (Late), Blue (Holiday)

**Assignment Submission Rate**
- Progress bar showing completion percentage
- Breakdown: Submitted, Pending, Late submissions

**Exam Performance Radar**
- Compare current scores vs previous scores vs class average
- Visualize performance across all subjects simultaneously

**Chapter Mastery Gauges**
- Individual cards for each chapter studied
- Proficiency levels: Mastered (80%+), Proficient (60-79%), Developing (40-59%), Needs Practice (<40%)

### Filtering Data

Use the date range selector at the top:
- **7 Days** - Last week's performance
- **30 Days** - Last month (default)
- **3 Months** - Quarter view
- **6 Months** - Half-year analysis
- **Custom** - Select specific date range

## For Teachers

### Accessing Class Analytics

1. Log in as a teacher
2. Navigate to **Dashboard** → **Analytics** or visit `/teacher/analytics/class/{classId}`
3. Select your class from the dropdown

### Available Features

**Class Score Trends**
- Track average, median, highest, and lowest scores over time
- Identify trends and patterns in class performance

**Student Distribution**
- Histogram showing how students are distributed across score ranges
- Helps identify if class is too easy/difficult

**Subject Difficulty Analysis**
- Expandable cards for each subject
- Shows difficulty level, pass rate, and common mistakes
- Identify subjects where students need more help

**Top Performers Table**
- See your top-performing students with their metrics
- Includes average score, attendance, and assignments completed

**Students Needing Support**
- Identify students who may need extra attention
- Shows weak subjects and attendance issues

### Taking Action

1. **Review Trends** - Look for declining performance patterns
2. **Analyze Distribution** - Ensure normal distribution of scores
3. **Address Difficulties** - Focus on subjects with low pass rates
4. **Support Struggling Students** - Reach out to students in the bottom performers list

## For Administrators

### Accessing Institution Analytics

1. Log in as an administrator
2. Navigate to **Admin** → **Analytics** or visit `/admin/analytics`
3. Choose from three tabs: Overview, Teacher Performance, Custom Reports

### Overview Tab

**Grade Comparison**
- Grouped bar chart comparing performance across grades
- Metrics: Average Score, Attendance Rate, Pass Rate

**Engagement Statistics**
- Key performance indicators with trend indicators
- Track overall institutional health

### Teacher Performance Tab

**Teacher Effectiveness Metrics**
- Comprehensive table of all teachers
- Metrics: Class Score, Student Satisfaction, Response Time, Assignments Graded
- Identify high performers and those needing support

### Custom Reports Tab

**Building Custom Reports**

1. **Select Date Range** - Choose start and end dates
2. **Filter by Grade** - Select one or multiple grades
3. **Filter by Section** - Choose specific sections
4. **Filter by Subject** - Select subjects of interest
5. **Choose Metrics** - Performance, Attendance, Assignments, etc.
6. **Group By** - Organize data by grade, section, subject, month, or week
7. **Click Generate** - Create your custom report

**Exporting Reports**

After generation:
- Click **Export PDF** - Download formatted PDF report
- Click **Export Excel** - Download data in Excel format for further analysis

## Common Use Cases

### Use Case 1: Student Progress Review
**Scenario:** Review a student's performance over the semester

1. Navigate to Student Analytics
2. Select **6 Months** date range
3. Review Subject Trends chart for overall progress
4. Check Attendance Heatmap for consistency
5. Examine Chapter Mastery to identify weak topics

### Use Case 2: Class Performance Assessment
**Scenario:** Evaluate how well a class is performing

1. Go to Class Performance Analytics
2. Review Class Score Trends for overall trajectory
3. Check Student Distribution - aim for normal curve
4. Analyze Subject Difficulty to identify challenges
5. Review Bottom Performers table for intervention

### Use Case 3: Monthly Report Generation
**Scenario:** Generate monthly performance report for board meeting

1. Open Institution Analytics Dashboard
2. Go to Custom Reports tab
3. Select last month's date range
4. Choose all grades and sections
5. Select all metrics
6. Group by Grade
7. Generate Report
8. Export to PDF for presentation

### Use Case 4: Teacher Performance Review
**Scenario:** Annual teacher performance evaluation

1. Navigate to Institution Analytics
2. Select Teacher Performance tab
3. Set date range to **1 Year**
4. Review Teacher Effectiveness Metrics table
5. Sort by different metrics (Class Score, Satisfaction, etc.)
6. Export data for detailed analysis

## Tips and Best Practices

### For Students
- Check analytics weekly to track progress
- Pay attention to declining trends in subject performance
- Use Chapter Mastery to prioritize study topics
- Maintain good attendance (75%+ recommended)
- Submit assignments on time to maintain submission rate

### For Teachers
- Review class analytics before parent-teacher meetings
- Use subject difficulty analysis to adjust teaching methods
- Check student distribution regularly to ensure appropriate difficulty
- Reach out to struggling students proactively
- Celebrate top performers to motivate class

### For Administrators
- Generate monthly reports for stakeholders
- Compare teacher effectiveness fairly (consider class composition)
- Track engagement statistics to measure institutional health
- Use grade comparisons to identify systemic issues
- Export data for detailed analysis in Excel

## Troubleshooting

### No Data Showing
- Ensure you have selected an appropriate date range
- Check if data exists for the selected period
- Verify you have the correct permissions

### Charts Not Loading
- Refresh the page
- Check internet connection
- Clear browser cache if issue persists

### Export Not Working
- Ensure report has been generated first
- Check browser's download settings
- Try a different browser if issue persists

### Performance Issues
- Reduce date range for faster loading
- Avoid selecting too many filters simultaneously
- Close other browser tabs to free memory
